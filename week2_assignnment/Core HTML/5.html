Some Web Dev Tech Stacks

MEAN Stack
A full-stack JavaScript framework known for building dynamic and robust web applications. It includes MongoDB for the database,
Express.js for the backend, Angular for the frontend, and Node.js as the runtime environment.

MERN Stack
A full-stack JavaScript framework known for building dynamic and robust web applications. It includes MongoDB for the database,
Express.js for the backend, Angular for the frontend, and Node.js as the runtime environment.

LAMP Stack
One of the oldest web development stacks, LAMP stands for Linux (operating system), Apache (web server), MySQL (database), and PHP (backend language).

Django Stack
A high-level Python web framework that encourages rapid development and clean, pragmatic design. It uses Django for the backend and can incorporate various frontend technologies.
