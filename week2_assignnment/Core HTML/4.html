Some HTML Tags

  1. <html>: Defines the root of an HTML document and encompasses all the content on the page.
  2. <head>:Contains meta-information about the document, such as its title, character encoding, and links to external resources.
  3. <body>: Contains the main content of the document, including text, images, links, and other elements visible to users.
  4. <a>: Creates a hyperlink, allowing users to navigate to another webpage or resource.
  5. <img>: Embeds an image into the document, enhancing visual content.
