HTML VS HTML 5

No. HTML HTML 5
1. In older versions of HTML (HTML 4 and earlier), the doctype declaration looked like this:
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN">; 		In HTML 5 the doctype declaration looks like this:
<!DOCTYPE HTML>
2. Limited multimedia support (requires plugins like Flash or Java) Native support for audio and video elements, as well as the <canvas> element for drawing graphics and animations
3. Limited semantic elements, such as div and span used for generic grouping Introduced new semantic elements like header, nav, article, section, footer, aside, figure, figcaption, mark, progress, output, and more, which provide more meaningful structure to the document.
